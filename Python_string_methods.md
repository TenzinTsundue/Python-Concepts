# Python string methods
1. Capitalize
```python
>>> name = 'tenzin'
>>> name_new = name.capitalize()
>>> name_new
'Tenzin'
```
2. casefold
```python
>>> txt = "One Does NOT simpLY Walk"
>>> x = txt.casefold()
>>> x
'one does not simply walk'
```
3. center
```python
>>> name = "Tenzin"
>>> name_new = name.center(10, "_")
>>> name_new
'__Tenzin__'
```
4. count
```python
>>> advice = "Watch The Office"
>>> f_count = advice.count('f')
>>> f_count
2
```
5. Encode
```python
>>> txt = "Mr Stale"
>>> new = txt.encode('ascii', 'ignore')
>>> new
b'Mr Stale'
```
6. Endswith
```python
>>> name = "Michael"
>>> end = name.endswith("ael")
>>> end
True
```
7. expandtabs
```python
>>> txt = "H\tT\tM\tL"
>>> x = txt.expandtabs(5)
>>> x
'H    T    M    L'
```
8. Find
```python
>>> txt = "pineapple on Pizza!"

>>> x

```
9. Format
```python
>>> txt = "I {} The {}".format("love", "Office")
>>> txt
'I love The Office'
>>> txt = "I {1} The {0}".format("Office", "love")
>>> txt
'I love The Office'
```
10. format_map 
```python
>>> kv = {'a' : 'Python', 'b': 'like'}
>>> x = 'I {b} {a}'.format_map(kv)
```
